# Prog1ATask1

QuickChat Java Application

Overview

QuickChat is a simple Java-based messaging application that allows users to register, log in, send messages, store or disregard messages, and manage message reports. This project covers three parts:
	•	Part 1: Registration and login system with username and password validation
	•	Part 2: Message sending with loops, numeric menu options, and message management
	•	Part 3: Storing messages in arrays, searching, deleting, displaying reports, and unit tests

⸻

How to Run
	1.	Open the project in NetBeans
	2.	Ensure JDK 11 or higher is installed
	3.	If using Maven, dependencies for JUnit4 are included in pom.xml
Features

Part 1 – Registration and Login
	•	Username validation (must include an underscore and ≤5 characters)
	•	Password complexity check
	•	Login authentication with confirmation and error messages
	•	Screenshots provided to demonstrate functionality
